# Heart Disease Exploratory Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on the Heart Disease dataset to understand the structure, distribution, and patterns of the data.

## Dataset
The dataset contains medical information about patients, including:
- Age
- Gender
- Cholesterol level
- Resting blood pressure
- Heart disease condition

## Analysis Performed
The following visualizations were created:
- Age distribution
- Gender distribution
- Cholesterol distribution
- Heart disease distribution
- Resting blood pressure distribution

## Conclusion
The analysis showed that most patients are between 50 and 65 years old, and there are more male patients than female patients. Cholesterol and blood pressure levels vary among patients.
